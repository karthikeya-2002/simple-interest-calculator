# Simple Interest Calculator

## Purpose
This repository contains a simple interest calculator script. The purpose of this project is to demonstrate basic Git and GitHub operations, including creating repositories, branching, and merging.

## How to use it
To calculate simple interest, you need three values:
1. **Principal amount (p)**
2. **Annual rate of interest (r)**
3. **Time period in years (t)**

The formula used is:
`Simple Interest = (p * r * t) / 100`

## Examples
If you borrow $1000 at a 5% interest rate for 2 years:
* p = 1000
* r = 5
* t = 2
* Simple Interest = (1000 * 5 * 2) / 100 = 100
